# CPF Validator

Esta biblioteca tem a funcionalidade de validar o número de CPF, para uso em aplicações web.

## Como instalar:
  `$  npm i amf-cpf-validator`
    
## Como utilizar:
```
  const { cpfValidator } = require("amf-cpf-validator");
  console.log(cpfValidator("889.571.648-55"));
```
